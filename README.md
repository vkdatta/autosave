#Implementation

``` javascript
<script src='https://cdn.jsdelivr.net/gh/vkdatta/autosave@main/autosave.lib.js'/>
```

#Usage

``` HTML
<input id="customername">ID tag is must for autosaving.</input>
```
Thats it! User can autosave this value on their localStorage
